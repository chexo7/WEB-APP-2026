# WEB-APP-2026

Aplicacion Next.js conectada a Firebase Realtime Database.

## Requisitos

- Node.js 20 o superior
- npm 10 o superior

## Puesta en marcha

```bash
npm install
npm run dev
```

La app usa la ruta `registros` dentro de Firebase Realtime Database.

## Variables de entorno

Las credenciales ya quedaron cargadas en `.env.local`. Si necesitas recrearlas, copia `.env.local.example` a `.env.local`.
